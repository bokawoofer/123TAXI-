# 123Taxi – Mobile PWA Starter
Tap-to-deploy taxi booking app for **123Taxi**.

## 1. Backend (Django + PostgreSQL)
[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/YOUR_USERNAME/123taxi)

Required env vars:
- `BUSINESS_NAME=123Taxi`
- `BUSINESS_PHONE=07717640052`
- `DRIVER_NAME=HOSSEIN`
- `STRIPE_PUBLISHABLE_KEY=pk_test_...`
- `STRIPE_SECRET_KEY=sk_test_...`
- `GOOGLE_KEY=...`

## 2. Frontend (Flutter PWA)
[![Deploy to Firebase Hosting](https://firebase.google.com/images/deploy/hosting/deploy-button.svg)](https://console.firebase.google.com/project/_/hosting/sites)

After Firebase finishes:
- Add your domain in Firebase → Hosting → Add custom domain.
- Install as PWA: open the URL → “Add to Home Screen”.

## 3. Quick Test
Use Stripe card `4242 4242 4242 4242` for success, `4000 0000 0000 9995` for failure.
